If you loaded this page from the web server running on localhost:9000, congratulations! You have successfully built and run a Play application.

Info on Project structure:

  • images at: public/images/
  • javascript files at: public/javascripts/
  • CSS at: public/stylesheets
  • all application routes to be mentioned at conf/routes
  • java files at app/controllers/
  • HTML files at app/views/ (they should be in format filename.scala.html)
